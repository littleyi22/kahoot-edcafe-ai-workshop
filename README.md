# 114 學年度 線上課程 Meet《Kahoot x EdCafe》研習工作坊網頁

本專案為奕鈞老師研習工作坊的專屬導覽與資源入口網站。提供參與研習的國中小教師一個清晰、流暢且具互動性的課程導覽與實戰素材庫。

## 🎯 專案特色
- **現代化介面設計**：深藍色科技感漸層背景、毛玻璃卡片（Glassmorphism）與懸停互動動畫。
- **完全響應式（RWD）**：針對手機直立窄版、iPad 直橫式以及桌上型電腦皆完成版面優化。
- **純前端架構（Zero Dependencies）**：僅使用 HTML / CSS / JS，無須編譯或安裝任何伺服器環境。
- **實戰素材一鍵複製**：內建《AI 演進史》文本與 10 題素養題庫，附帶 JavaScript 一鍵複製功能，方便教師快速建立教材。
- **AI 備課工具整合**：收錄 PIRLS 選擇題、是非題、素養題等命題機器人快捷連結。

## 🚀 執行與啟動方式
本網頁為純靜態檔案，無須伺服器環境。
1. 對著 `index.html` **點擊兩下**，即可在任何現代瀏覽器（Chrome, Edge, Safari 等）中直接開啟並檢視。
